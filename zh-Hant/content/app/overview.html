概要

這個 app 是為了讓 blackjack 學習不只停留在策略表，而是能透過練習、統計與回放，把算牌流程和長期結果一起看懂。

學習算牌時，常見問題不是只知道 Hi-Lo 牌值，而是能不能在接近實戰的情境中穩定追蹤 Running Count、換算 True Count、依照 TC 調整下注，並回頭檢討錯誤。除了練習之外，app 也能用大量模擬和統計結果，幫助理解 EV、ROI、變異數、bet ramp 和 session 規則如何影響 blackjack 的長期表現。

主要功能

策略與偏離練習

練習基本策略、硬牌、軟牌、對子、加倍、投降、保險，以及算牌偏離。

RC / TC 訓練

透過 Running Count 和 True Count 練習，熟悉從牌序追蹤到下注區間判斷的流程。

算牌系統

支援 Hi-Lo、KO、Red 7、FELT、Zen、Hi-Opt I、Omega II 七套系統，可搭配不同桌規、牌組數、穿透率與 bet ramp 測試。

策略自訂

針對每套算牌系統建立自訂 deviation 規則集，修改 index 門檻或移除特定規則，搭配遊戲與決策訓練驗證差異。

回放與統計

用 Replay 檢討過去決策，並透過大量模擬觀察 ROI、EV、SD、bankroll、停利停損和長期結果。

學習路線

  1. 先用 Decision Training 把基本策略、投降、保險、對子和常見 deviation 練穩，確認自己不是只看得懂策略表，而是能在題目中做出一致決策。
  2. 接著用 Running Count Drill 練 Hi-Lo 累加速度，再用 True Count Drill 練 RC 和剩餘牌數的換算，讓 count 不只停留在概念，而是能快速變成下注判斷。熟悉後可切換到其他算牌系統，各系統均有獨立的訓練入口。
  3. 再進入 Count Drill 和 Game 模式，在多人發牌、不同速度與完整牌桌流程中追蹤 count，並練習依照 True Count 和 Bet Ramp 調整下注。
  4. 玩完後用 Replay 檢查錯誤，特別是下注、動作與 insurance 的 Mistakes，把常犯問題抓出來，再回到對應的訓練模式補強。
  5. 最後用 Round Stats 和 Session Stats 測試規則、Bet Ramp、停損停利和不同策略的長期差異，理解 EV、ROI 和變異數，而不是只看單次輸贏。
目標不是把 blackjack 做成休閒遊戲，而是提供一個能反覆練習、觀察數據、修正錯誤，並用統計理解遊戲波動的算牌訓練工具。
