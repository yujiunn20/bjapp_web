Training

Training mode breaks practice into four independent modes targeting decision accuracy, Running Count speed, True Count conversion, and full-session count tracking that simulates real table pacing. All seven counting systems are available in Count Drill.

Decision Training

Focuses on blackjack decisions — basic strategy and counting Deviations.

  • Mode Switch: Choose Basic, Hi-Lo, or Hi-Opt I to practice the same hand under different judgment standards.
  • Question Type: Select specific categories — Hard Totals, Soft Totals, Pairs, Doubles, Surrender, Insurance, Deviations — or mix them.
  • Instant Feedback: Shows whether the answer was correct and the strategy reason immediately after each choice.

The Deviations category targets count-dependent departures from basic strategy, useful for drilling rare situations that don't come up often in live play.

Running Count Drill

Practices Hi-Lo accumulation speed as cards are dealt in sequence.

  • Configurable deal speed and sequence length — start short and slow, then increase both.
  • Starting RC can be fixed at 0 or randomized to practice picking up from mid-count.
  • Display Mode switches between one card at a time, two cards, or random-size groups to simulate real table rhythm.
  • After answering, all previously shown cards are displayed so you can trace where the count drifted.

True Count Drill

Practices converting Running Count and remaining cards into True Count.

  • Each question gives an RC and remaining card count; you calculate the True Count mentally.
  • Repeated practice with different RC / deck combinations builds speed and accuracy for real-time bet decisions at the table.

Count Drill

The practice mode closest to live play. Simulates dealer and multi-player dealing in sequence so you track the count throughout a full hand rhythm.

  • Each session starts with a given RC and remaining cards, then simulates a hand at the configured player count and speed.
  • At the end, you answer the final RC or remaining deck count to confirm you can track stably under near-live pacing.
  • Player count and deal speed are adjustable.
  • Supports all seven counting systems: Hi-Lo, KO, Red 7, FELT, Zen, Hi-Opt I, Omega II — each with its own drill entry.
Suggested practice order: stabilize basic strategy and Deviations with Decision Training → build RC speed with RC Drill → practice TC conversion with TC Drill → integrate everything under full-hand rhythm in Count Drill.
