Privacy Policy

Blackjack Card Counting Trainer does not collect, store, or transmit any personal data. This policy explains how the application handles data.

No Personal Data Collected

This application:

  • Requires no account or personal information to use.
  • Does not collect names, email addresses, device identifiers, or any personally identifiable information.
  • Contains no advertising tracking or usage analytics.

Data Stored Locally

The following data is stored only on the user's local device and is never sent to any external server:

  • Play logs: Per-round records from Play mode, stored as .jsonl.gz files in the local backend log directory.
  • Simulation results: Monte Carlo simulation output files, also stored as .jsonl.gz files in the local log directory.
  • App settings: Table rules, Bet Ramp, counting system preferences, and other settings, stored in the local app data directory.
  • Strategy Profiles: User-created Basic Strategy Tables and Deviation rule sets, stored locally on the device.
  • PRO unlock status: After a PRO purchase through Microsoft Store, the unlock status is recorded locally on the device.

All of the above data is under the user's full control. Individual log files can be deleted through the Files section in Stats, or the entire app data directory can be removed directly through the file system.

Network Connections

The application starts a local Python backend service (running on localhost) so the UI can communicate with the game engine. This connection is entirely local and does not go through the internet.

Microsoft Store handles network communication for app installation, updates, and PRO plan purchases. This is managed by Microsoft and is outside the control of this application.

Third-Party Services

This application is distributed through the Microsoft Store; installation and purchases are handled by the Microsoft platform. Microsoft's privacy policy applies to interactions with Microsoft Store — please refer to the Microsoft Privacy Statement for details.

No other third-party analytics, advertising, or social services are integrated into this application.

No Real-Money Gambling

This application is an educational training tool. It does not include real-money gambling, online betting, or in-app virtual currency purchases. All in-app "bets" and "chips" are training metrics only and have no real monetary value.

Changes to This Policy

If this Privacy Policy is updated significantly, the content on this page will be revised and the date below will be updated accordingly.
