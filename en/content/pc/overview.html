Overview

Blackjack Card Counting Trainer PC is a Windows desktop training tool available on the Microsoft Store. Powered by a Python simulation engine, it covers Play, Training, Sim & Stats, Replay, and Strategies, with support for seven counting systems: Hi-Lo, KO, Red 7, FELT, Zen, Hi-Opt I, and Omega II.

Feature Overview

Play

A realistic blackjack table with up to seven player seats. Displays Running Count and True Count in real time, with toggleable bet recommendations and decision feedback.

Training

Four practice modes: Decision Training, Running Count Drill, True Count Drill, and Count Drill — switchable across all seven counting systems.

Sim & Stats

Python-backed Monte Carlo simulation. Configure hand count, table rules, counting system, and Bet Ramp to generate long-term performance data.

Replay

Review Game session records. Filter by bet, action, or Insurance mistakes, and step through each hand via Timeline to inspect decision details.

Strategies

Build custom Deviation rule sets for eight systems (Basic, Hi-Lo, Hi-Opt I, KO, Red 7, FELT, Zen, Omega II). Adjust Index thresholds and apply them to Game and Training.

Learn

Rules reference, basic strategy charts, and counting notes as a study resource.

Counting Systems

Training mode supports seven counting systems, each with a dedicated Count Drill entry and Strategy Profile:

  • Hi-Lo (FREE)
  • KO
  • Red 7
  • FELT
  • Zen
  • Hi-Opt I
  • Omega II

FREE vs PRO

The FREE tier includes Hi-Lo counting and core features. Unlocking PRO via in-app purchase on the Microsoft Store adds:

  • Up to seven player seats (FREE is limited to three)
  • Custom deck count, penetration, and H17 rules
  • Custom Bet Ramp thresholds and multipliers
  • Hi-Opt I Strategy Profile customization

Platform

Windows 10 / 11, installed via the Microsoft Store. Interface supports Traditional Chinese, English, and Japanese.
