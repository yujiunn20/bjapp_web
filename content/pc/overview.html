概要

二十一點算牌訓練器 PC 版是一套執行於 Windows 的桌機訓練工具，可從 Microsoft Store 下載。以 Python 為模擬核心，提供遊戲、訓練、模擬統計、回放與策略自訂，支援 Hi-Lo、KO、Red 7、FELT、Zen、Hi-Opt I、Omega II 共七套算牌系統。

功能一覽

遊戲（Play）

接近實戰的 blackjack 牌桌，最多七個玩家席位。即時顯示 Running Count 與 True Count，可開關建議下注與決策對錯提示。

訓練（Training）

四種練習模式：Decision Training、Running Count Drill、True Count Drill、Count Drill，七套系統均可切換。

模擬統計（Sim & Stats）

Python 後端 Monte Carlo 模擬，可設定局數、桌規、算牌系統與 Bet Ramp，生成長期表現數據。

回放（Replay）

回放 Game 遊玩紀錄，可篩選下注、動作、Insurance 錯誤，並用 Timeline 逐手查看決策細節。

策略自訂（Strategies）

針對八套系統（Basic、Hi-Lo、Hi-Opt I、KO、Red 7、FELT、Zen、Omega II）建立自訂 Deviation 規則集，修改 Index 門檻後套用至 Game 與 Training。

學習（Learn）

規則說明、基本策略表與算牌筆記，作為學習參考。

算牌系統

訓練模式支援七套算牌系統，每套均有獨立的 Count Drill 入口與 Strategy Profile 設定：

  • Hi-Lo（FREE）
  • KO
  • Red 7
  • FELT
  • Zen
  • Hi-Opt I
  • Omega II

FREE 版與 PRO 版

FREE 版包含 Hi-Lo 算牌與基本功能。透過 Microsoft Store 內購解鎖 PRO 版後，可解鎖以下功能：

  • 最多七個玩家席位（FREE 限三個）
  • 自訂牌組數、穿透率與 H17 桌規
  • 自訂 Bet Ramp 門檻與下注倍率
  • Hi-Opt I 策略自訂（Strategy Profiles）

平台

Windows 10 / 11，透過 Microsoft Store 安裝。介面支援繁體中文、English 與日本語。
