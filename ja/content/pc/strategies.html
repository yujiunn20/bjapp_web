戦略カスタム

Strategies は PRO 版機能で、各カウントシステムごとに Basic Strategy Table と Deviation ルールセットをカスタマイズし、Play・Training・Simulation の判断に適用できます。

Strategy Profile 概要

左のサイドバーでカウントシステム（Basic・Hi-Lo・KO・Red 7・FELT・Zen・Hi-Opt I・Omega II）とディーラールール（H17 / S17）を絞り込むと、該当する Profile がリストに表示されます。

アプリには読み取り専用の Preset Profile（各システム 1 つ）が付属しています。カスタマイズするには Duplicate でコピーするか、+ Add Custom で新規作成してください。Custom Profile のみ編集・保存・削除が可能です。

Strategy Table のカスタマイズ

各 Profile には Hard Totals・Soft Totals・Pairs の 3 セクションからなる Basic Strategy Table が含まれます。横軸はディーラーのアップカード（2–10・A）、縦軸はプレイヤーのハンド値またはペア種別で、各セルにはその状況の標準アクションが表示されます。

Custom Profile ではセルのドロップダウンからアクション（Hit / Stand / Double / Split / Surrender）を変更できます。変更済みのセルはゴールドでハイライトされます。Clear Overrides をクリックすると、すべての変更を一度にデフォルトに戻せます。

Deviation ルールのカスタマイズ

テーブル下の Deviation セクションにすべてのカスタムルールが一覧表示されます。ルールの追加・有効 / 無効切り替え・削除が可能です。各ルールには以下を設定します：

  • Hand Type：Hard / Soft / Pair / Insurance。
  • Player value vs Dealer upcard：このルールが適用されるハンドとディーラーのアップカードの組み合わせ。ワイルドカード * 使用可。
  • Trigger：カウントシグナルのしきい値（>= / > / <= / < + 値）。均衡システム（Hi-Lo・FELT・Zen・Hi-Opt I・Omega II）は True Count、KO / Red 7 は Running Count を使用。
  • Action：シグナルがしきい値に達したときのアクション（Hit / Stand / Double / Split / Surrender / Insurance）。
  • 有効 / 無効：ルールごとに個別に切り替え可能。削除せず無効化できます。
  • Notes：自由記述のメモフィールド。

適用範囲

保存した Custom Profile は以下の 3 か所で選択できます：

  • Play モード：Settings で各席に Strategy Profile を割り当てると、Play のベット推奨と判断の正誤フィードバックがカスタム Table と Deviation ルールに基づいて判定されます。
  • Training — Decision Training：Strategy Profile を選択すると、Deviation 問題がカスタムルールに従って出題・採点され、自作の Deviation を集中練習できます。
  • Simulation（PRO）：Sim 設定で Strategy Profile を選ぶと、カスタム戦略でシミュレーションを実行し、長期的な EV と ROI を評価できます。
Strategies は PRO 版機能です。FREE ユーザーは Preset Profile の内容を閲覧できますが、Custom Profile の作成や設定の変更はできません。
