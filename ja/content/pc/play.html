プレイ

Play モードは実戦に近いブラックジャックテーブルを提供し、最大 7 席（FREE 版は 3 席）まで設定できます。カウント HUD・ベット推奨・判断提案・正誤フィードバックを画面に統合し、完全なテーブルリズムの中で基本戦略とカウントを同時に練習できます。

テーブル機能

  • カウント HUD：画面上部に Running Count（RC）と True Count（TC）をリアルタイム表示。現在の Mode（Basic・Hi-Lo・Hi-Opt I）に応じたカウント情報を切り替えて表示します。
  • ベット推奨：「Use Recommended」をタップすると、現在のカウントシステム・True Count・Bet Ramp に基づいた推奨ベットを自動入力。手動入力も可能。
  • 判断提案：有効にすると、各ハンドの判断前に推奨アクション（Hit・Stand・Double・Split・Surrender・Insurance）と戦略理由を表示。
  • 正誤フィードバック：各判断後に現在の Mode 基準と照合した結果を即座に表示し、基本戦略の安定確認に役立ちます。
  • マルチシート：最大 7 席を設定し、自分の座席を選択。実際の多人数テーブルのリズムに近い環境でトレーニングできます。

プレイログ（Play Log）

ゲーム画面上部の Play log コントロールで、各セッションの手牌を保存するかどうかを選択できます。保存した記録は Replay で後から確認できます：

  • off：記録しない — 手牌は保存されません。
  • new：新規ファイル — セッションごとに新しい .jsonl.gz ログファイルを作成します。
  • append：既存ファイルに追記 — ドロップダウンから以前のログを選択し、新しい手牌をその後ろに追加。複数セッションにわたるトレーニング履歴を継続的に積み上げられます。

ゲーム中の Strategy Table

プレイ中、ゲーム画面右上の Strategy Table アイコンをクリックすると、戦略参照パネルが開きます。パネルには現在のテーブルルールに対応した Basic Strategy 表（ハード / ソフトトータル）と現在のカウントシステムの Deviation ルールが表示され、Strategies で設定したカスタム Index しきい値やアクションも含まれます。プレイ中に推奨の根拠をすぐに確認できます。

設定オプション

右上の設定パネルでゲームを中断せずに以下の項目を調整できます：

  • Mode：Basic・Hi-Lo・Hi-Opt I を切り替え。ベット推奨・判断提案・正誤判定の基準が変わります。
  • テーブルルール：H17 / S17・Double ルール・Surrender・Insurance など、さまざまなカジノ条件を再現。
  • デッキ数とペネトレーション：使用デッキ数とカット深度を調整。True Count 換算とカウント難易度に影響します。
  • Bet Ramp：True Count しきい値と対応するベット倍率を設定。推奨ベット計算と Replay でのベットミス判定に影響します。
  • 配牌速度：スローからファストまで調整可能。最初はゆっくり計算時間を確保し、慣れてきたら速度を上げて実戦プレッシャーを体験。
  • 席とプレイヤーモード：プレイヤー数・自分の席・各席の Mode（Basic・Hi-Lo・Hi-Opt I）を設定して、さまざまな練習シナリオを構築。
  • Hi-Opt I 設定（PRO）：Ace サイドカウント関連の高度なオプション。
配牌速度を上げると、実戦に近いペースで RC / TC 計算とベット判断を同時にこなす必要が生まれ、安定した習慣を身につける効果的な練習になります。
