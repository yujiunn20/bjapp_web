Play

Play mode provides a realistic blackjack table with up to seven player seats (three in FREE). The screen integrates a count HUD, bet recommendations, decision suggestions, and correctness feedback so you can practice both basic strategy and card counting in a full table rhythm.

Table Features

  • Count HUD: Displays Running Count (RC) and True Count (TC) at the top of the screen in real time. Updates according to the current Mode (Basic, Hi-Lo, Hi-Opt I).
  • Bet Recommendation: Tap "Use Recommended" to have the app calculate the suggested bet based on the current counting system, True Count, and Bet Ramp, or set the bet manually.
  • Decision Suggestion: When enabled, shows the recommended action (Hit, Stand, Double, Split, Surrender, Insurance) and the strategy reason before each decision.
  • Correctness Feedback: After each decision, immediately shows whether the choice matched the current mode's standard, helping confirm basic strategy consistency.
  • Multi-Seat: Configure up to seven player seats and choose your own position to simulate a realistic multi-player table rhythm.

Settings

The settings panel in the top-right corner can be adjusted without interrupting the game:

  • Mode: Switch between Basic, Hi-Lo, or Hi-Opt I to determine which standard governs bet recommendations, decision suggestions, and correctness checks.
  • Table Rules: H17 / S17, Double rules, Surrender, Insurance — simulate different casino conditions.
  • Deck Count & Penetration: Adjust the number of decks and cut depth, affecting True Count conversion and counting difficulty.
  • Bet Ramp: Set True Count thresholds and corresponding bet multipliers, affecting recommended bet calculations and Replay's bet-mistake judgments.
  • Deal Speed: From slow to fast. Start slow to allow time for RC / TC calculation; increase speed to simulate real-table pressure.
  • Seats & Player Modes: Set the number of players, your own seat, and the Mode (Basic, Hi-Lo, Hi-Opt I) for each seat to build different practice scenarios.
  • Hi-Opt I Settings (PRO): Advanced options for Ace side count.
Increasing deal speed requires completing RC / TC calculation and bet judgment within a pace closer to live play — an effective way to build stable habits under pressure.
