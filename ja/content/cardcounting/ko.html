補足：KO

KO（Knock-Out）は不平衡なカードカウントシステムです。最大の特徴は True Count への換算が不要で、Running Count を直接ベットや deviation の判断に使えるため、Hi-Lo より実戦での操作が簡単です。

Hi-Lo は平衡システムで、1 デッキ分をカウントすると RC が 0 に戻るため、残りデッキ数で割って True Count に換算する必要があります。KO は 7 を +1 にカウントするため、1 デッキあたり +4 の余剰が生じ、デッキをカウントし終えても RC は 0 に戻りません。この「不平衡」によって RC 自体がデッキの残り枚数の情報を持つため、割り算の換算なしに RC のしきい値で直接判断できます。

Hi-Lo との違いは 7 が 0 から +1 になる点だけです。他のカード値はまったく同じです。

IRC（Initial Running Count）

KO は不平衡システムのため、1 デッキ分のカウント後の RC は 0 ではなく +4 になります。異なるデッキ数でも RC しきい値が一貫した意味を持つよう、IRC を起点として設定します。IRC = -4 × デッキ数で、6 デッキの場合は IRC = -24 となり、不平衡による影響を相殺するように起点 RC をマイナス方向にシフトします。

アプリの KO トレーニングモードでは起点 RC の設定が可能で、さまざまなデッキ数での練習に対応しています。

KO は True Count 換算が不要なため実戦のハードルが低く、割り算なしにカードの追跡だけに集中したいプレイヤーに向いています。代わりに、Hi-Lo の TC 調整と比べると一部の判断精度が落ちます。特にデッキが深いまたは浅い場合に誤差が大きくなります。
