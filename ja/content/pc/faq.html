よくある質問

ここでは PC アプリの目的、モバイルとの違い、対応カウントシステム、FREE vs PRO、Sim、Strategies 機能、データ管理、動作環境を整理しています。

PC アプリはモバイル アプリと何が違いますか？

PC アプリは Windows 専用のトレーニングプラットフォームで、モバイルアプリと同じ blackjack 学習の目標を共有しながら、モバイルにはないいくつかの機能を追加しています。主な違いとして、Python を使ったモンテカルロシミュレーションエンジン（深い統計分析）、7 つの完全なカウントシステム（Hi-Lo・KO・Red 7・FELT・Zen は FREE；Hi-Opt I・Omega II は PRO が必要）、最大 7 つのプレイヤー席、各システムのカスタム deviation ルールセットを作成できる Strategies 機能、ローカルに .jsonl.gz 形式で保存されるゲーム記録があります。

どのカウントシステムに対応していますか？

合計 8 つのモードに対応しています：Basic（基本戦略のみ）、Hi-Lo・KO・Red 7・FELT・Zen（いずれも FREE）、Hi-Opt I・Omega II（PRO が必要）。各システムに独立した Decision Training、Count Drill、Bet Ramp、deviation index、Strategies のカスタマイズが用意されています。Hi-Opt I は Side A / Side B の ace side count 設定もサポートしており、より正確なベットとインシュランス判断が可能です。

FREE と PRO の違いは何ですか？

FREE には完全な blackjack ゲーム、Hi-Lo・KO・Red 7・FELT・Zen の Decision Training と Count Drill、Strategy Profile カスタマイズ、Hi-Lo の Bet Ramp 設定、テーブルルールのカスタマイズ、Sim エンジン（最大 10 万手）が含まれます。PRO では Hi-Opt I と Omega II のカウントシステム、Hi-Opt I の Side A / Side B ace side count、Hi-Opt I と Omega II の Strategy Profile カスタマイズ、7 つのカウントシステム全てに対応した Bet Ramp カスタマイズ、シミュレーション上限の拡張（最大 500 万手）が利用可能になります。

Sim 機能はどのように動作しますか？

Sim は Windows マシン上でローカルに動作する Python Monte Carlo シミュレーションエンジンを使用します。テーブルルール、カウントシステム、Bet Ramp、ラウンド数を設定してシミュレーションを実行すると、長期 EV、ROI、SD / 100 を取得できます。FREE は最大 10 万手、PRO は最大 500 万手まで対応しており、理論期待値により近い結果を得られます。システムや設定間で結果を比較することもできます。Python バックエンドは完全ローカルで動作し、データは外部サーバーには送信されません。

Strategies 機能とは何ですか？

Strategies では、8 つのシステム（Basic と 7 つのカウントシステム）それぞれに対してカスタム deviation ルールセットを作成・管理できます。各 Strategy Profile 内で個々の deviation の index しきい値を調整できます。Strategy Profile をゲームやトレーニングセッションに適用すると、アプリはデフォルトの代わりにカスタムルールを使用します。異なる index 値のテストや特定のハウスルールへの対応に役立ちます。

データはどこに保存されますか？

すべてのゲーム記録は .jsonl.gz 形式の圧縮ファイルとして Windows マシンにローカル保存されます。Python シミュレーションバックエンドも完全にローカルで動作します。個人データ、ゲーム記録、シミュレーション結果は外部サーバーにはアップロードされません。

PC アプリにインターネット接続は必要ですか？

必要ありません。ゲーム、トレーニング、シミュレーション、Replay、Strategies 機能はすべてオフラインで動作します。インターネット接続が必要なのは、初めて Microsoft Store からアプリを購入またはインストールするときと、Microsoft Store のライセンス確認のときだけです。

必要な Windows バージョンは何ですか？

Windows 10 または Windows 11 が必要です。アプリは Microsoft Store を通じて配布されています。

カウントすれば必ず勝てますか？

いいえ。カウントは、ルール、ペネトレーション、ベットスプレッド、実行精度が適切な場合に長期期待値を改善できる可能性があるだけです。長期 EV が良くなっても、短期結果は分散に大きく左右されます。実戦では資金、テーブルリミット、心理的プレッシャー、ゲーム速度、カジノ側の対策なども問題になります。

このアプリはギャンブルを勧めるものですか？

いいえ。このアプリは blackjack の確率、戦略、カウント概念を学ぶための教育ツールです。リアルマネーのギャンブル機能はなく、カウントを確実に稼げる方法として扱っていません。利用者は所在地の法律とプラットフォーム規約を守り、blackjack にはリスク、短期変動、資金面のプレッシャーがあることを理解する必要があります。
