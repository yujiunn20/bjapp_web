プライバシーポリシー

Blackjack Card Counting Trainer は、個人情報の収集・保存・送信を一切行いません。本ポリシーではアプリのデータ取り扱い方針を説明します。

個人情報の非収集

本アプリは：

  • アカウント登録や個人情報の提供なしに利用できます。
  • 氏名・メールアドレス・端末識別子など、個人を特定できる情報を収集しません。
  • 広告トラッキングや利用状況の分析機能を含みません。

ローカルに保存されるデータ

以下のデータはユーザーの端末内にのみ保存され、外部サーバーへ送信されることはありません：

  • プレイログ：Play モードで記録されるラウンドごとのデータ。.jsonl.gz 形式でローカルバックエンドのログディレクトリに保存。
  • シミュレーション結果：Simulation が生成するモンテカルロ結果ファイル。同じく .jsonl.gz 形式でローカルログディレクトリに保存。
  • アプリ設定：テーブルルール・Bet Ramp・カウントシステムなどの設定情報。ローカルアプリデータディレクトリに保存。
  • Strategy Profile：ユーザーが作成した Basic Strategy Table と Deviation ルールセット。端末内に保存。
  • PRO 解錠状態：Microsoft Store での購入後、解錠状態が端末内に記録されます。

上記データはすべてユーザーが管理できます。Stats の Files セクションから個別のログファイルを削除するか、ファイルシステムからアプリデータディレクトリを直接削除することが可能です。

ネットワーク接続

アプリは UI とゲームエンジンが通信するためのローカル Python バックエンドサービス（localhost で動作）を起動します。この接続は完全にローカルで行われ、インターネットを経由しません。

Microsoft Store は、インストール・アップデート・PRO プランの購入時にネットワーク通信を行います。これは Microsoft が管理するものであり、本アプリの制御範囲外です。

サードパーティサービス

本アプリは Microsoft Store を通じて配信されており、インストールと購入は Microsoft プラットフォームが処理します。Microsoft Store との連携には Microsoft のプライバシーポリシーが適用されます。詳細は Microsoft のプライバシーに関する声明をご参照ください。

本アプリにはその他のサードパーティ分析・広告・ソーシャルサービスは一切統合されていません。

実際の賭博機能なし

本アプリは教育目的のトレーニングツールです。実際の賭博・オンラインベッティング・仮想通貨のアプリ内購入機能は含まれていません。アプリ内の「ベット」や「チップ」はトレーニング指標であり、実際の金銭的価値はありません。
