Strategies

The Strategies page lets you build custom Deviation rule sets for eight counting systems. Adjust the Index threshold (True Count trigger) for each rule and apply the customized set to Play and Training decision judgment.

Strategy Profiles

The app provides eight Strategy Profiles, one per judgment standard:

  • Basic (no counting)
  • Hi-Lo
  • Hi-Opt I (PRO only)
  • KO
  • Red 7
  • FELT
  • Zen
  • Omega II

Each Profile stores its own Deviation rule set independently.

Deviation Rule Customization

Within each Profile, each Deviation rule can be:

  • Enabled / Disabled: Choose whether the rule is included in Play and Training judgment.
  • Index Modified: Adjust the True Count threshold that triggers the Deviation — for example, changing the Insurance threshold from +3 to +2, or adjusting the Stand threshold for 16 vs 10.

After saving, Play mode's decision suggestions and Decision Training's Deviation questions will use the customized rule set for the corresponding system.

Where Changes Apply

  • Play Mode: When Mode is set to the corresponding system, bet recommendations and correctness feedback use the custom Index thresholds.
  • Decision Training: Deviation question types are generated and scored according to the custom rules.
Hi-Opt I Strategy Profile customization (including Ace side count options) is a PRO-only feature. Rule set customization for the other seven systems is available in the FREE tier.
